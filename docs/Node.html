Class: Node

Node(label, textnullable)

A node inside a CPEE process tree parsed from an XML document. A node is considered a(n) - leaf node if it corresponds to a activity DSL-Element. - inner node if it corresponds to a control flow DSL-Element. - property node otherwise. In the case that a node is the root node, i.e. does not have a parent, it is labelled as a tree in code.
